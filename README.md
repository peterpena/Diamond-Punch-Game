**1. How To Play**
===================

The goal of the game is to score 5 baskets before the other player

**2. Controls**
================

Player 1
------------
'Q'    - Grab Diamond

'E'    - Throw Diamond

'W'    - Up

'S'    - Down

'A'    - Left

'D'    - Right

Player 2
-------------------

'Shift'         - Grab Diamond

'Return'        - Throw Diamond

Up Arrow Key    - Up

Down Arrow Key  - Down

Left Arrow Key  - Left

Right Arrow Key - Right
